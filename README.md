### Установка и запуск

###### Зависимости
```
& pip install PyQt5
& pip install sqlalchemy
& pip install pycryptodome
& pip install pycryptodomex
```
###### Запуск

Для начала работы необходимо запустить сервер
```
$ cd server
$ python server.py
```
После чего уже можно слушать и принимать клиентов
```
$ cd /client
$ python client.py
```
